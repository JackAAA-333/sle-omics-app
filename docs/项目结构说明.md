# SLE分析 项目结构说明

## 1) 主要入口

- `全流程非线性自动组学分析/app.py`：前端页面入口（Streamlit）
- `全流程非线性自动组学分析/run_pipeline.py`：上传数据后的一键流程调度
- `streamlit_app.py`：云端部署入口（Streamlit Community Cloud）

## 2) 核心分析脚本（项目根目录）

- `preprocess_and_analyze.py`：代谢组分析
- `proteomics_analysis.py`：蛋白组分析
- `multiomics_integration.py`：双组学整合分析
- `generate_reports.py`：汇总报告生成

## 3) 应用与打包相关

- `全流程非线性自动组学分析/start_app.sh`：本地静默启动脚本
- `全流程非线性自动组学分析/desktop_launcher.py`：桌面窗口启动逻辑
- `全流程非线性自动组学分析/logo.icns`：应用图标

## 4) 文档目录

- `docs/`：说明文档与使用手册
